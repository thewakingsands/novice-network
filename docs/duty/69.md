# 阿修罗歼灭战

粉丝节试玩的蛮神讨伐战，于6.55版本实装于游戏中。

开场出啥躲啥，召唤分身会使用圆形AOE，远离所有分身。

BOSS召唤出场边大号分身之后，会随机读条使用：
* 剑技转写·圆斩：自身圆形AOE
* 剑技转写·环斩：自身环形AOE
* 剑技转写·剑光波：点名随机一名玩家直线AOE

之后大号分身会复读BOSS技能，其中直线AOE为连线BOSS本体的超宽直线，<Role name="tank" /><Role name="healer" /><Role name="dps" />所有人需要到场地两边躲避。

读条==三头六臂：断==后按顺序出现6次半场AOE预告，预告结束后会按预告顺序释放半场AOE。

读条==愤怒的神气==或==愉悦的神气==，同时场地变为红蓝亮色，出现气泡和画面特效。读条愤怒/红色特效表示红色半场AOE，读条愉悦/蓝色特效表示蓝色半场AOE。

之后出啥躲啥，大号分身再次复读之前会随机传送到场地另一位置，需要观察大分身传送方向进行躲避。
